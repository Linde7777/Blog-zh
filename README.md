# 文学类
《麦田里的守望者》  
第一人称视角讲述一个辍学高中生在外面到处逛的故事，我读了两三遍也没悟出什么大道理来，但就是很喜欢。
英文原版的词汇不难。
  
《百年孤独》  
  现实+魔幻主义，总体情节还算有意思，看的时候记得梳理一下人物关系，不然到后面谁是谁都分不出来
    
《牧羊少年的奇幻之旅》  
  还行吧  
    
儒勒·凡尔纳三部曲《格兰特船长的儿女》《海底两万里》《神秘岛》  
  不知道是不是因为翻译的问题，有些描述有点幼稚，像是那种十二三岁的人写的东西，不过瑕不掩瑜，情节不错。  
  
  《雾都孤儿》  
  一般般，不推荐
