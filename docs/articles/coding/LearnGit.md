补充：
1.进入你的github主页，点击右上角头像，点击settings，点击左侧的Respositories，把Repositories default brach的名字从 main 改为 master  
2.在VS Code中，用Ctrl+`打开终端，用Ctrl+J关闭终端，你可以在终端中输入git指令，也可以用VS Code里面的git插件
